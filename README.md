Internet Radio Project with Ai Thinker ESP32 Audio KIT rev 2748 PCB(AC101 CODEC)

The contents of the SD Card Files.zip need to be unpacked on a SD card.

If you failed to connect to the network then broadcast your own WiFi under ESP32WEBRADIO (Password: ESP32PSWD)

Key3 [PREVIOUS] Previous radio station

Key4 [NEXT] Next radio station

Key5 [VOLUME DOWN] Adjust volume -

Key6 [VOLUME_UP] Adjust volume +


The ORIGINAL project with ES8388 CODEC from: https://github.com/thieu-b55/ESP32-audiokit-webradio-webinterface/blob/main/ESP32_webradio_audiokit.ino   Copyright (c) 2022 thieu-b55


I modified thieu-b55's project to work with the ac101 audio codec, and added the buttons to use it.
